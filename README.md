# SwiftCartE-Commerce

1. null এবং undefined এর মধ্যে পার্থক্য কী?
   a. হলো একটি মান যা ইচ্ছাকৃতভাবে খালি বা শূন্য হিসেবে সেট করা হয়।
   b. বোঝায় যে কোনো ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু তাকে কোনো মান দেওয়া হয়নি।

2. JavaScript-এ map() ফাংশনের ব্যবহার কী? এটি forEach() থেকে কীভাবে আলাদা?
   a. একটি নতুন অ্যারে তৈরি করে যেখানে প্রতিটি উপাদানকে নির্দিষ্ট ফাংশনের মাধ্যমে পরিবর্তন করা হয়।
   b. শুধু প্রতিটি উপাদানের উপর কাজ করে কিন্তু নতুন অ্যারে রিটার্ন করে না.

3. == এবং === এর মধ্যে পার্থক্য কী?
   a. শুধু মান তুলনা করে, টাইপ চেক করে না।
   b. মান এবং টাইপ দুটোই তুলনা করে।
   উদাহরণ: → true, কিন্তু → false।
4. API ডেটা ফেচ করার সময় async/await এর গুরুত্ব কী?
   a. ব্যবহার করলে কোড আরও পড়তে সহজ হয় এবং অ্যাসিঙ্ক্রোনাস কাজগুলোকে সিঙ্ক্রোনাসের মতো লেখা যায়।
   v. এটি প্রমিস হ্যান্ডলিংকে সহজ করে এবং API থেকে ডেটা আসা পর্যন্ত অপেক্ষা করতে দেয়।
5. JavaScript-এ Scope (Global, Function, Block) ধারণাটি ব্যাখ্যা করো।
   a. Global Scope: যেসব ভ্যারিয়েবল পুরো প্রোগ্রামে অ্যাক্সেসযোগ্য।
   b. Function Scope: ফাংশনের ভিতরে ডিক্লেয়ার করা ভ্যারিয়েবল শুধু সেই ফাংশনের ভিতরে অ্যাক্সেসযোগ্য।
   c. Block Scope: ব্লকের ভিতরে ডিক্লেয়ার করা ভ্যারিয়েবল (যেমন , ) শুধু সেই ব্লকের ভিতরে অ্যাক্সেসযোগ্য।
